# FlexTalk
FlexTalk은 대용량 트래픽을 경험해보기 위해 시작하는 토이프로젝트입니다.

기술스택은 다음과 같습니다.

FrontEnd
 - html, css, javascript, react, redux, redux-saga 

BackEnd
 - spring, mysql, redis, netty, kafka, jpa, querydsl, junit, mockito
